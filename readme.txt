== Installation ==

After you've downloaded and extracted the files:

1. Upload the complete 'easy-location' folder to the '/wp-content/plugins/' directory OR install via the plugin installer
2. Activate the plugin through the 'Plugins' menu in WordPress
3. And yer done!
